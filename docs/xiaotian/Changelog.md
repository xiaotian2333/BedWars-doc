# 更新日志

## 版本号规则说明

版本号规则：主版本号.次版本号.修正版本号

一般情况下主版本号将保持为1。除非底层大重构，否则此版本号不会改变

次版本号将在有新功能加入时更新，且此时修正版本号归零

修正版本号将在修复bug或新增黑名单等对玩法没有改变时更新
